# Example Notes

This directory contains example markdown notes for testing the Obsidian Actions GitHub Action.

## Test Notes

- `example-note.md` - Has `example` tag, should match example filter
- `important-note.md` - Has `important` tag, won't match example filter
- `drafts/draft-note.md` - Draft note in subdirectory
- `published-article.md` - Has `example` tag and additional properties
- `simple-note.md` - No frontmatter, but has inline `#example` tag

These notes can be used to test various filter configurations and script executions.
